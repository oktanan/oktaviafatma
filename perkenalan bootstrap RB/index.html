Aboutme

OKTAVIA FATMAWATI

Haii, perkenalkan nama saya Oktavia Fatmawati, biasa dipanggil okta,okto atau fatma. Saya lahir di Karanganyar, 29 Oktober 2005, alamat rumah Bendo 01/09, Pojok, Mojogedang, Karanganyar. Saya pelajar SMKN 2 Karanganyar yang beralamat di JL. Yos Sudarso. Saya masih duduk di kelas XI, saya mengambil jurusan Rekayasa Perangkat Lunak. Awalnya saya ingin masuk SMKN 1 Karanganyar dengan jurusan Akuntansi tetapi nilai saya tidak mencukupi untuk masuk Sekolah tersebut. Alhamdulillah saya masih diterima disekolahan negeri supaya tidak membebani orang tua saya. Saya memiliki banyak impian dan harapan begitupun dengan orang tua saya. Harapan-harapan saya antara lain :

  • Sukses
  • Bahagia Dunia Akhirat
  • Membahagiakan Orang tua
